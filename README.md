# Что это?
В данном репозитории хранятся исходные коды [данного сайта](https://nightmarescult.github.io/algocourses-solutions) с решениями контестов с t-edu.tech в параллели C (2025-2026). Решения выкладываются сугубо после дедлайна (так как мы не поддерживаем списывание и бездумную копипасту). 

# Сборка
```sh
# вам нужен cargo
cargo install mdbook
git clone https://github.com/nightmarescult/algocourses-solutions/ dest
cd dest
mdbook build
```
